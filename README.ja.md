# panelist-helper
English README is available here: [README.md](README.md)

パネル討論の際にパネリストを支援するため、ディスカッションに基づいて有益なコメントを提供するツールです。

## デモ
[ライブデモ](https://js.sabae.cc/panelist-helper.html)

## 機能
- 登壇者のコメントを音声認識で捕捉する
- ディスカッションに基づいて役立つコメントを自動生成する
- 日本語と英語のいずれかを選択できる

## 要件
このツールには、Web Speech APIをサポートする最新のWebブラウザが必要です。

## 使い方
1. 互換性のあるブラウザで[デモページ](https://js.sabae.cc/panelist-helper.html)を開きます。
2. ドロップダウンから希望の言語(日本語または英語)を選択します。
3. "音声認識の開始"ボタンをクリックして始めます。
4. ツールは"認識結果"テキストエリアであなたの発話を書き起こします。
5. "ヘルプフルなコメント"テキストエリアは、書き起こされた発話に基づいて関連するコメントを自動的に提案します。
